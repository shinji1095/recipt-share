# Summary
レシートの金額をLINEでシェアするためのチャットBOTアプリケーション．Firestoreにてレシートを管理．REST APIでデータを保存，取得，削除する．

# Requirements
- Flask
- Guicorn
- line-bot-sdk
- firebase-admin
- python-dotenv

# コミットメッセージの命名規則

| コメントのprefix | 説明 |
| ---------------- | --- |
| add | 新規機能追加 |
| fix | バグ修正 |
| update | 機能修正（バグではない）|
| remove | 削除（ファイル）|
